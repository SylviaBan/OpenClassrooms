Choix de la métrique ¶

C'est le score F1 que nous allons utiliser comme m√©trique d'√©valuation. C'est la moyenne pond√©r√©e de la pr√©cision et du rappel. Ce score prend en compte √† la fois les faux positifs et les faux n√©gatifs.

  • F1 Score = 2(Recall * Precision) / (Recall + Precision)
  • Recall = TP/TP+FN
  • Precision = TP/TP+FP

En effet, l'id√©e est de d√©tecter un maximum de tweets n√©gatifs. Tout en sachant que :

  • Les vrais n√©gatifs sont labelis√©s √† 0 (TN) => il faut augmenter la Pr√©cision
  • Les faux positifs sont labelis√©s √† 1 (FN) => il faut √©galement augmenter le Recall afin d'identifier un maximum de vrais positifs.

Nous allons surveiller √©galement l'aire sous la courbe ROC (AUROC), le temps √©coul√© lors de l'entra√Ænement (Time) d'un mod√®le et l'√©volution de la perte (Loss).

def model_Evaluate(model):

    # Predict values for Test dataset
    y_pred = model.predict(X_test)
    # Print the evaluation metrics for the dataset.
    print(classification_report(y_test, y_pred))
    
    # Compute and plot the Confusion matrix
    cf_matrix = confusion_matrix(y_test, y_pred)
    categories = ['Negative', 'Positive']
    group_names = ['True Neg', 'False Pos', 'False Neg', 'True Pos']
    group_percentages = ['{0:.2%}'.format(
        value) for value in cf_matrix.flatten() / np.sum(cf_matrix)]
    labels = [f'{v1}\n{v2}' for v1, v2 in zip(group_names, group_percentages)]
    labels = np.asarray(labels).reshape(2, 2)
    
    fig, ax = plt.subplots(figsize=(8,6))
    sns.heatmap(cf_matrix, annot=labels, cmap='Oranges', fmt='',
                xticklabels=categories, yticklabels=categories, ax=ax)
    plt.xlabel("Predicted values", fontdict={'size': 14}, labelpad=10)
    plt.ylabel("Actual values", fontdict={'size': 14}, labelpad=10)
    plt.title("Confusion Matrix", fontdict={'size': 18}, pad=20)
